ECA-DISC



En este proyecto, desarrollo Full-Stack de una aplicación móvil diseñada para personas con necesidades especiales. Este proyecto fue patrocinado por la Fundación Chilena para la Discapacidad. La aplicación está diseñada para facilitar la comunicación y la coordinación entre la empresa, las personas con discapacidad y sus tutores. Las tecnologías empleadas incluyen React.js y React Native para la interfaz de usuario, TypeScript para una mejor gestión de tipos en el código, Node.js para el servidor backend, y Firebase para la base de datos y autenticación.

Kingdoms of Ereloth



En este proyecto, me encargué del desarrollo Full-Stack utilizando Blueprints en Unreal Engine para Arcadium, una empresa especializada en la creación de videojuegos. El juego es un RPG (Role-Playing Game) que se desarrolla en un universo de fantasía. Los jugadores tienen la capacidad de interactuar con el entorno y con otros personajes dentro del juego. Las tecnologías empleadas incluyen diversas versiones de Unreal Engine (5.0, 5.1, 5.2) para el motor del juego, Blueprints para la programación visual y diseño de efectos visuales (VFX) para mejorar la experiencia del usuario.
